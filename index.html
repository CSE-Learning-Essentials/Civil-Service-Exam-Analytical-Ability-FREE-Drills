SECTION 2: ANALYTICAL ABILITY

PART 3: LOGICAL REASONING

PRACTICE TEST 1 | 5-ITEM TEST | PASSING SCORE: 3

Direction: Each item below is given with statements. Identify the best conclusion from the choices that logically follows the given statements.

1. Statements: All Greeks are philosophers. Socrates is a Greek.

Conclusion:
I. All Greeks are Socrates
II. Socrates is a philosopher.
III. Some Socrates is philosophers.
IV. Some philosophers are Socrates.

2. Statements: Some actors are singers. All the singers are dancers.

Conclusion:
I. Some actors are dancers.
II. No singer is actor.

3. Statements: All deer have divided-hoofs. All goats have cloven-hoofs.

Conclusion:
I. All deer have cloven-hoofs.
II. All goats have divided-hoofs
III. Some deer have cloven-hoofs.
IV. Some goats have divided-hoofs.

4. Statements: All mammals are animals. All cats are mammals.

Conclusion:
I. All mammals are cats.
II. All cats are animals.
III. Some mammals are cats.
IV. No cats are animals

5. Statements: Some ships are boats. All boats are submarines. Some submarines are yatches.

Conclusion:
I. Some yatches are boats.
II. Some submarines are boats.
III. Some submarines are ships.
IV. Some yatches are ships

Correct Answers:

  1. B. II only — Socrates is a Greek, and all Greeks are philosophers → Socrates is a philosopher.
  2. A. Only I conclusion follows — Some actors are singers and all singers are dancers, so some actors could be dancers.
  3. E. No valid conclusion — No direct relationship can be established between deer and cloven-hoofs or goats and divided-hoofs.
  4. B. II only — All cats are mammals and all mammals are animals → All cats are animals.
  5. B. Only II and III follow — From the statements, some submarines are boats, and some submarines could be ships.
