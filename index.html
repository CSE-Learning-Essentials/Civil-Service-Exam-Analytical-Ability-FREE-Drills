PRACTICE TEST | 5-ITEM TEST | PASSING SCORE: 3

Directions: Read each question and choose the correct answer.

1. If the day before yesterday was Wednesday, then which day comes one day after tomorrow?

2. Which day comes 1 day after Sunday?

3. If the day before yesterday was the 23rd, then what will be the day after tomorrow?

4. The day before two days after the day before tomorrow is Saturday. What day is it today?

5. If the day before yesterday was Thursday. When will Sunday be?

Explanations:

  1. C. Sunday
    Day before yesterday = Wednesday → Yesterday = Thursday → Today = Friday → Tomorrow = Saturday → One day after tomorrow = Sunday.
  2. A. Monday
    One day after Sunday is clearly Monday.
  3. C. The 27th
    Day before yesterday = 23rd → Today = 25th → Day after tomorrow = 27th.
  4. D. Friday
    "Day before tomorrow" = today → “Two days after today” = one day after tomorrow = Saturday, so today = Friday.
  5. A. Tomorrow
    Day before yesterday = Thursday → Today = Saturday → Tomorrow = Sunday.
